JARVIS : The Desktop Assistant

A desktop AI assistant featuring voice command processing, speech recognition, and Gemini support for hands-free desktop automation.
Overview

This desktop assistant is inspired by the Avengers movies and powered by Gemini. Designed to streamline desktop automation, it executes complex workflows through simple voice or text commands. For example, when instructed to 'create an essay on [topic],' the assistant automatically searches Google, compiles the relevant information, writes it into a Notepad file, and saves it directly to the desktop. It can also set timers and answer general knowledge questions. While the current version relies on code generated by ChatGPT and Gemini, I am actively working to refine and improve its capabilities.

I played no role in developing this project; I simply copied and pasted the code from ChatGPT and Gemini.
