Voice Controlled Robotic Car

An electric car model that can be controlled in real time using voice commands from a mobile phone.
Overview

This electric car model is controlled via real-time voice commands from a mobile phone, which serves as the system's primary processing unit for speech recognition. When a voice command like 'Forward' is spoken, the phone converts the audio into text and transmits it to the car via an HC-05 Bluetooth module. The Arduino UNO, acting as the onboard microcontroller, receives this signal and processes it according to the pre-programmed logic. It then sends the appropriate control signals to the L298N motor driver, which regulates the power and direction of the motors to move the car.
