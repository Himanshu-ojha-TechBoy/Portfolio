AI Chatbot

JARVIS-inspired AI assistant built via MIT App Inventor, providing Gemini-powered answers alongside voice-activated mobile calling, messaging, and music playback.
Overview

This project is a mobile-based AI assistant built using MIT App Inventor. Inspired by Iron Man’s JARVIS, the application features a futuristic, high-tech UI/UX and integrates continuous speech recognition alongside text-to-speech capabilities. Powered by the Gemini API, the assistant functions as an interactive chatbot that can answer general queries and perform basic mobile automation tasks.Key FeaturesVoice-Controlled Automation: Users can execute hands-free commands. For instance, saying "Call Manish" automatically triggers a phone call, while saying "Send to Karan [message]" dispatches a text to the specified contact.Gemini AI Chatbot Integration: Handles natural language queries and answers simple user questions through AI processing.JARVIS-Inspired UI/UX: Designed with a sleek, interactive, and high-tech visual theme reminiscent of sci-fi interfaces.Media Playback: Capable of playing selected songs upon voice request.Development & CollaborationSelf-Developed: Independently designed the entire UI/UX, calling systems, and message-automation framework with minimal AI guidance.AI-Assisted: Leveraged ChatGPT and Gemini to generate, debug, and implement the backend integration code for the Gemini API.
